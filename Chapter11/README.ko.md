## 단계별 개발

### 1단계: 기본 HTML 만들기

```
> index.html에 기본 구조 만들어줘.
> 헤더, 메인, 푸터 섹션으로 나눠줘.
```

Claude가 이런 구조를 만들어줍니다:

```html
<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>내 포트폴리오</title>
    <link href="style.css" rel="stylesheet">
</head>
<body>
    <header>
        <!-- 네비게이션 -->
    </header>

    <main>
        <!-- 메인 컨텐츠 -->
    </main>

    <footer>
        <!-- 푸터 -->
    </footer>

    <script src="script.js"></script>
</body>
</html>
```

### 2단계: 자기소개 섹션

```
> 자기소개 섹션 추가해줘.
> - 프로필 사진 자리
> - 이름
> - 한 줄 소개
> - 간단한 자기소개 문단
```

### 3단계: 프로젝트 갤러리

```
> 프로젝트 섹션 추가해줘.
> - 카드 형태로 3개 프로젝트
> - 각 카드에 썸네일, 제목, 설명
> - 호버하면 살짝 올라오는 효과
```

### 4단계: 연락처 폼

```
> 연락처 섹션 추가해줘.
> - 이름 입력
> - 이메일 입력
> - 메시지 입력
> - 보내기 버튼
```

### 5단계: 스타일링

```
> Tailwind CSS 적용해줘.
> 깔끔하고 현대적인 느낌으로.
> 색상은 파란색 계열로.
```

### 6단계: 반응형 만들기

```
> 모바일에서도 잘 보이게 반응형으로 바꿔줘.
> 작은 화면에서는 메뉴가 햄버거로 바뀌게.
```

## 피드백으로 개선하기

첫 결과가 마음에 안 들어도 괜찮습니다. 피드백을 주세요.

### 예시

```
> 헤더가 너무 커. 높이 줄여줘.

> 버튼 색이 별로야. 더 진한 파란색으로.

> 글씨가 너무 작아. 크기 키워줘.

> 프로젝트 카드 사이 간격 넓혀줘.
```

2-3번 피드백 주면 원하는 디자인이 나옵니다.
