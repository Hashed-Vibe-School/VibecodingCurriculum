## Common Commands Reference

```bash
# Installation
curl -fsSL https://claude.ai/install.sh | bash

# Basic usage
claude                          # Interactive session
claude "prompt"                 # One-shot command
claude -p "prompt"              # Print mode (non-interactive)

# Session management
claude --continue               # Continue last session
claude --resume                 # Choose from past sessions

# Permission modes
claude --permission-mode plan        # Read-only
claude --permission-mode acceptEdits # Auto-approve edits

# Model selection
claude --model claude-sonnet-4-5-20250929
claude --model claude-haiku
```

## Key Shortcuts
| Shortcut | Action |
|----------|--------|
| `Esc Esc` | Undo to last checkpoint |
| `Shift+Tab` | Cycle permission modes |
| `Alt+P` / `Option+P` | Switch model |
| `Ctrl+C` | Cancel current action |
| `Ctrl+B` | Send to background |
